# ─── M-WESTY CLOTHID  ───

Uma solução sofisticada e intuitiva de captura de componentes de vestuário (IDs de roupas) para FiveM, desenvolvida com foco em alta performance e refinamento visual para desenvolvedores e designers de moda.

### Especificações

* **Compatibilidade** · Totalmente Standalone (compatível com qualquer base ou framework).
* **Performance** · Estrutura leve e otimizada (baixo consumo de 0.00ms constante no resmon).
* **Instalação** · Processo simplificado estilo "drag and drop", sem dependências complexas.

### Funcionalidades

* **Captura de IDs** · Leitura em tempo real de todos os 15 slots de vestuário e texturas do personagem.
* **Controle de Câmera** · Sistema inteligente de foco rápido (cabeça, tronco, pernas, pés) com rotação do ped e zoom.
* **Exportação Rápida** · Botões dedicados para copiar os dados já formatados em Tabela Lua ou JSON diretamente para a área de transferência.
