# M-Westy ClothID - Captura de IDs de Roupas

---

## 📌 SOBRE O RECURSO
O **M-Westy ClothID** é uma ferramenta premium desenvolvida para simplificar a vida de administradores, designers de roupas e desenvolvedores. Ele abre um painel intuitivo e fluido para capturar os IDs de todas as roupas que seu personagem está vestindo atualmente, permitindo exportar os valores formatados diretamente para copiar no formato Lua ou JSON.
